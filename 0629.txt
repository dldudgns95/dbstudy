관리자 계정
1. SYS, SYSTEM 계정이다.
2. 관리자 계정으로 접속해서 수업에서 사용할 새로운 계정을 만든다.
3. 관리자 계정으로 작업하지 않도록 주의한다.

새로운 계정 만드는 방법
1.CREATE USER 아이디 IDENTIFIED BY 비밀번호;
2.GRANT CONNECT TO 아이디;

기존계정 삭제 방법
1.계정이 가진 데이터가 없을 때 : DROP USER 아이디;
2.계정이 가진 데이터가 있을 때 : DROP USER 아이디 CASCADE;
